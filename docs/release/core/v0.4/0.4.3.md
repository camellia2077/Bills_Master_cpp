## [v0.4.3] - 2026-03-26

### 技术改进/重构 (Changed/Refactor)
* 重构 IO 模块目录与文档入口，将 `libs/bills_io` 迁移到 `libs/io`、将 `docs/modules/bills_io` 迁移到 `docs/modules/io`，统一仓库内的模块导航口径。
* 重构 CLI、Android、toolchain 与 boundary include allowlist 的 IO 路径引用，将源码 include、构建入口、notices 路径和扫描配置统一切换到 `libs/io` / `docs/modules/io` / `io/...`。
* 重构终端颜色常量归属，将颜色定义从 `libs/bills_core/src/common/common_utils.hpp` 迁移到 CLI 展示层，避免 `bills_core` 承担终端呈现职责。

### 弃用/删除 (Deprecated/Removed)
* 删除 `libs/bills_io/` 与 `docs/modules/bills_io/` 旧目录入口，移除仓库中遗留的旧 IO 模块路径引用。
* 删除 `libs/bills_core/src/common/common_utils.hpp`，移除核心库内的 ANSI 颜色常量定义。
