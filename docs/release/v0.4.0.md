## [v0.4.0] - 2026-03-05

### 新增功能 (Added)
* 新增 `tools/verify/tools/check_import_layering.py`，为调用层建立 `import` 优先门禁，并支持 `--stats` 与白名单基线维护。
* 新增 `tools/verify/tools/check_boundary_layering.py`，为边界层建立独立门禁，强制每条 quoted include 绑定 `owner/reason/window/tier`。
* 新增 `tools/verify/tools/config/boundary_include_whitelist.json`，固化边界层长期项与可替换项分层策略。
* 新增 `libs/bills_core/src/modules/phase2_call_chain_module_import_smoke.cpp`，覆盖 `application/reports/abi` 三条调用链导入冒烟。
* 新增 `apps/bills_cli/src/common/cli_version.hpp`，实现 **CLI** 版本与 **Core** 版本独立维护。

### 技术改进/重构 (Changed/Refactor)
* 重构 `apps/bills_cli/src/windows/presentation/cli/controllers` 与 `command_handler` 调用层依赖，统一走 `import` 与 `ports` 抽象，收敛直连实现头。
* 重构 **AppController/WorkflowController/ExportController** 组合根装配路径，移除对具体序列化器、格式化器、文件处理器的调用层直连。
* 重构 `libs/bills_core/src/abi/internal/abi_shared.hpp` 边界聚合入口，收敛 `convert/ingest/query/validate` handlers 的 OFF 通道回退 include。
* 重构 `libs/bills_core/src/common/version.hpp`，核心版本常量统一迁移到 `bills::core::version` 命名空间并升级到 `0.4.0`。
* 重构 `libs/bills_core/src/modules/common_version.cppm`，模块导出版本改为透传 `bills::core::version`，保持 ON/OFF 双通道一致。
* 重构 `apps/bills_cli/src/windows/presentation/cli/controllers/app_controller.cpp` 的 `display_version()`，统一输出 `Core Version` 与 `CLI Version` 双版本信息。
* 推进目录分层：新增 `libs/bills_io/` 独立 IO 适配层并由 `apps/bills_cli` 作为组合根注入，稳定 `ports` 契约方向。

### 修复 (Fixed)
* 修复 `libs/bills_core/src/billing/conversion/modifier/processor/bills_parser.cpp` 的静态 `regex` 生命周期与浮点比较告警，改为显式 `YYYY-M` 解析与有序比较分支。
* 修复 `libs/bills_core/src/serialization/bills_json_serializer.cpp` 的静态 `regex` 生命周期告警，改为显式 `YYYY-MM` 解析。
* 修复 `libs/bills_core/src/billing/conversion/validator/bills_validator.hpp` 头部冗余依赖，收敛 `regex` 依赖到实现文件，降低包含噪音。
* 修复调用层白名单陈旧项：清理 `tools/verify/tools/config/import_layer_whitelist.json` 中 handlers 的 8 条 stale 记录。
* 修复边界层“无主 include”风险：边界白名单条目全部补齐 `owner/reason/window/tier` 元数据。

### 弃用/删除 (Deprecated/Removed)
* 删除 `apps/bills_cli/src` 迁移后空目录骨架（`adapters/serialization`、`db_insert/*`、`reports/core|monthly_report|yearly_report`）。
* 删除调用链过渡接口 `handle_modification`（`AppController` 与 `WorkflowController`），`modify` 命令统一回归 `handle_convert`。
* 删除调用层遗留业务 include 直连（含 `ports/bills_repository.hpp`、reports contracts 直连头），持续压缩 include 白名单规模。

### 验证 (Verification)

* 通过 `python tools/verify/verify.py core-abi`，ABI 边界行为保持稳定。
* 通过 `python tools/verify/verify.py` 全量验证（构建 + CLI 测试）。
