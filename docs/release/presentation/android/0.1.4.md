## [v0.1.4] - 2026-04-01

### 新增功能 (Added)
* 新增 `Editor` 界面账单自动开启逻辑。现在点击年份或月份下拉列表中的选项后将直接加载对应内容，无需手动点击确认。
* 新增 `Workspace -> Data` 中的 `Export TXT + Config ZIP` 按钮，可将 `records/` 下的账单文本与配置文件一起导出为 ZIP 包。

### 技术改进/重构 (Changed/Refactor)
* 调整 `Editor` 界面中 `Open Existing Period` 的下拉列表排序规则，改为按日期升序排列（旧日期在前，新日期在后）。
* 移除 `Editor` 界面中冗余的 `Open Existing` 按钮，简化交互流程。
* 复用 `libs/io` 中的 `ExportParseBundle` 导出链路，为 Android 侧统一导出 `TXT + Config` ZIP 能力，避免重复维护独立打包逻辑。

### 修复 (Fixed)
* 修复在 `Editor` 界面保存新月份账单后，`Query` 界面的月份下拉列表未及时同步刷新新日期的问题。
