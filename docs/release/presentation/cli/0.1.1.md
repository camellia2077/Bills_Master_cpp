## [v0.1.1] - 2026-03-26

### 新增功能 (Added)
* 新增 `report export` 成功提示，导出完成后打印成功导出的报告数量和导出目录。
* 新增 `report export --format` 多格式列表支持，允许使用 `md,json,tex,typ` 这样的逗号分隔格式覆盖导出配置。

### 技术改进/重构 (Changed/Refactor)
* 调整 `report export` 的格式解析优先级：显式传入 `--format` 时优先使用命令参数，未传入时回退到 `config/export_formats.toml` 的 `enabled_formats`。
* 重构 CLI 终端颜色常量定义，将颜色实现收敛到 `apps/bills_cli/src/presentation/common/terminal_colors.hpp` 并统一由 CLI 展示层使用。
