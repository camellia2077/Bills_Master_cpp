## [v0.4.1] - 2026-03-06

### 技术改进/重构 (Changed/Refactor)
* 重构 `libs/bills_io/src/bills_io/adapters/config/`，将配置加载链路统一替换为 `Toml*` 实现，配置读取改为直接从 **TOML** 解析到结构体，并移除 `apps/bills_cli` 下已脱离构建的旧配置适配器残留。
* 重构 `config/` 与 `tests/config/` 下的配置资源分发方式，将 `export_formats`、`modifier_config`、`validator_config` 统一切换为 `.toml` 文件。
* 重构 `apps/bills_cli/cmake/source_files.cmake`、`libs/bills_core/cmake/source_files.cmake` 与 `libs/bills_io/CMakeLists.txt`，同步接入新的 **TOML** 配置适配器与内建报表 formatter 装配。
* 重构 `libs/bills_core/src/reports/formatters/` 目录结构，将原 `apps/bills_cli/src/reports/plugins/` 下的月报与年报 formatter 下沉到 **Core** 层集中维护。
* 重构 `libs/bills_io/src/bills_io/adapters/reports/` 与 `libs/bills_io/src/bills_io/io_factory.*`，以 `builtin_*_report_formatter_provider` 替代动态插件 provider，统一由内建 formatter 提供 `md/rst/tex` 能力。
* 重构 `apps/bills_cli/src/windows/presentation/cli/controllers/app_controller.cpp` 与 `export/export_controller.cpp`，移除报表插件扫描与文件注入逻辑，导出链路改为按当前构建内建格式判定可用性。
* 重构 `tools/flows/bills_tracer_flow.py` 与 `tests/framework/_test_internal/*.py`，移除 `[plugins]`、`plugin_targets`、`plugin_dlls` 等测试链路概念，改为内建 formatter 构建与运行模型。
* 重构 `docs/modules/bills_io/architecture.md`、`docs/modules/bills_io/change_guide.md` 与 `docs/modules/bills_io/module_map.md`，模块文档口径同步更新为 `adapters/config` 使用 **TOML**、`adapters/reports` 使用内建 formatter。
* 同步 `libs/bills_core/src/common/version.hpp`，将核心版本升级到 `0.4.1`。

### 修复 (Fixed)
* 修复 `libs/bills_core/src/billing/conversion/modifier/processor/bills_parser.cpp`、`libs/bills_core/src/billing/conversion/validator/bills_validator.cpp` 与相关生成链路的月份格式约束，账单头部日期统一收敛为严格 `date:YYYY-MM`。
* 修复 `tests/generators/log_generator/src/internal/bill_generator.cpp` 生成策略与 CLI 测试断言，单月账单输出改为 ISO 月格式，避免 `YYYY-M` 与 `YYYY-MM` 混用。
* 修复 `tests/generators/log_generator/`、`tests/suites/logic/log_generator_cli/run_tests.py` 与 `tools/flows/log_generator_build/artifact_pipeline.py` 的配置口径，`log_generator` 运行时默认改为读取 `config.toml`。
* 修复 `libs/bills_core/src/ports/config_provider.hpp` 与 `libs/bills_core/src/application/use_cases/workflow_use_case.hpp` 的 `Result.hpp` include 大小写不一致问题，消除 Windows/Clang 的非便携路径告警。

### 弃用/删除 (Deprecated/Removed)
* 删除 `apps/bills_cli/src/reports/plugins/` 下的旧报表 formatter 源码、空目录骨架与对应的 CMake 入口，停止从 CLI 层维护报表实现。
* 删除 `libs/bills_io/src/bills_io/adapters/plugins/common/` 下的动态插件加载器与旧 `dynamic_*_report_formatter_provider` 实现，报表导出不再依赖 DLL 插件。
* 删除 `apps/bills_cli/src/config/*.json`、`tests/config/*.json` 与 `libs/bills_io/src/bills_io/adapters/config/json_*` 旧 JSON 配置资源和加载实现。
