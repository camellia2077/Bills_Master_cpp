# bills_io Docs

这是 `libs/bills_io` 的模块文档入口。目标是让 agent 先知道“应该看哪个目录、在哪一层改、去哪里 rg”。

## Read In This Order

1. `libs/bills_io/README.md`
2. `libs/bills_io/AGENTS.md`
3. `docs/modules/bills_io/module_map.md`
4. `docs/modules/bills_io/change_guide.md`
5. 需要看整体边界时再看 `docs/modules/bills_io/architecture.md`
